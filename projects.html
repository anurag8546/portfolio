PROJECTS

⚫ React TODO App


Tech used: HTML, CSS, JavaScript, React.js
Link: TODO-App


Features


1) This is an offline app for the browsers built using Reactjs library.
2) Users can add their TODOs locally on the browser.
3) These TODOs can be updated and deleted.
4) Hard refresh (reloading the page) doesn't affect the current state of the application.
5) On clicking on Reset button, all the entries are wiped off.

⚫ Quiz Application using C++


Tech used: C++, File Handling
Link: Quiz App


Features


1) A program that reads 10 random questions from a question booklet.
2) User gets 4 options for each question, one out of which is correct.
3) After completion of test, a detailed result is shown to the user.
4) The result includes, the correct number of questions, right answer for each question and responses of user and the percentage score of the user.
